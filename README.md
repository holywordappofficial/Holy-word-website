## Quick Start

### Frontend
```bash
cd client
npm install
npm start
```

### Backend
```bash
cd server
npm install
npm start
```

## API Endpoints

- `GET /api/daily-verse` - Today's verse
- `GET /api/verses` - All verses
- `GET /health` - Server status
